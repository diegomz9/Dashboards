## Overview
This project analyzes the **market performance of 10 major U.S. companies** using Power BI, focusing on **returns, trading volume, price behavior, volatility, and sector dynamics** over a one-year period.

The goal of this dashboard is not only to visualize financial data, but to **extract actionable insights** that explain *why* certain companies outperformed others and *how* investor behavior differed across sectors.

This project was built as a **portfolio piece for recruiters and hiring managers**, demonstrating both technical Power BI skills and analytical storytelling.
